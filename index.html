Lombokeats - App | Status : Active

This project focuses on the development of a mobile web application aimed at recognizing and promoting Lombok Island's unique gastronomic heritage. Utilizing a Deep Learning model with Siamese Neural Network architecture, users can easily capture images of traditional Lombok dishes through their smartphone or photo gallery. These images are sent to a server where they are identified using a pre-trained model. The application then provides detailed gastronomic information about the recognized dish, including historical and cultural insights.

Gapulo - Backend | Status : Active

This project focuses on the development of the Gastronomi Backend & Admin App, a robust backend system built with Laravel and powered by a PostgreeSQL database. It is designed to support the gastronomic food recognition platform on Lombok Island, offering seamless integration with the frontend via RESTful APIs. This system allows for efficient data communication between the mobile frontend and backend server, ensuring that users can explore Lombok's rich culinary heritage.

SIM-AMI - App | Status : Active

The Internal Quality Audit Information System (SIM-AMI) is a platform developed to streamline the internal quality audit process in higher education institutions, specifically implemented in Politeknik Elektronika Negeri Surabaya. It addresses challenges in manual monitoring, spreadsheet-based data management, and inefficiencies in report generation. The system adheres to Ministerial Regulation No. 53 of 2023 and focuses on auditing performance indicators (IKUK) while improving data management and ensuring compliance with regulations.
